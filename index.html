What is a "Circle of Suck"?

A circle of suck is when there is a circle of teams or things being better than each other in cyclical manner. For example, 'A' is better than 'B', and 'B' is better than 'C', yet 'C' is better than 'A'. There is no clear "winner" as to who or what is best.

How to Use

Upload a .json file with the contents of the file such as follows. Read this format as "A is better than B and C, B is better than C and D..."

{
    "A":["B","C",],
    "B":["C","D"],
    "C":["D"],
    "D":["A"] 
}
